Reminders app made within React framework

App features:

  • Click (+) to add a new reminder
  • Click (trash) to delete a task
  • Click (bell) or double click the reminder to toggle the priority of the reminder

Front end of app only. Starting data is loaded as default appwide state which is reset on reload.

Possibility of writing Redux methods and connecting with state to extend the funcationality of the app
